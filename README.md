## Project Overview
Surtika is a modern e-commerce platform for curated Indian fashion. Built with a focus on rich aesthetics, 3D interactions, and a seamless shopping experience.

### Tech Stack
- **Frontend:** HTML, Vanilla CSS, JavaScript
- **Backend:** Firebase (Auth, Firestore, Functions)
- **Design System:** Stitch (Customized for Surtika)

## Local Development
1. Clone the repository.
2. Run a local server (e.g., `python3 -m http.server 5500`).
3. Update `firebase-config.js` with your credentials.
